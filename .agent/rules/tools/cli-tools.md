 
# CLI Tools

## 1. Docker
- Commands:
  - `docker ps` → List running containers
  - `docker compose up -d` → Start project containers
  - `docker compose down` → Stop containers
- Usage: "List running containers"

## 2. Git
- Commands:
  - `git status` → Show repo status
  - `git add . && git commit -m "msg"` → Commit changes
- Usage: "Commit latest changes to repo"

> Guidelines:
> - Only suggest commands from approved CLI tools.
