 
# Database Tools

## 1. MySQL
- Purpose: Query project database
- Input: SQL query
- Output: Table result in markdown
- Example: "SELECT * FROM users WHERE role='admin';"

## 2. MongoDB
- Purpose: Fetch documents or update collections
- Input: JSON query
- Output: JSON data
- Example: "Find all users with subscription=true"

## 3. PostgreSQL
- Purpose: Query or manage PostgreSQL project database
- Input: SQL query
- Output: Table result in markdown
- Example: "SELECT email FROM clients WHERE active=true;"


> Guidelines:
> - Only use sanitized queries.
> - Avoid destructive operations unless explicitly authorized.
